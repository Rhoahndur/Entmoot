- postgres_data/                        # Database persistence

KEY TECHNOLOGIES:
=================
Backend:  FastAPI, GDAL, Shapely, GeoPandas, PyProj, Rasterio
Frontend: React 19, TypeScript, Vite, TailwindCSS, MapLibre GL
Database: PostgreSQL 15 + PostGIS 3.4
Cache:    Redis 7
DevOps:   Docker, Docker Compose, GitHub Actions
Testing:  pytest, coverage

DEPLOYMENT OPTIONS:
===================
1. Local Development:  docker compose -f docker-compose.dev.yml up
2. Production:         docker compose up -d
3. CI/CD:             Automatic via GitHub Actions

QUICK START:
============
$ cp .env.example .env
$ docker compose up -d
$ open http://localhost

For more details, see DOCKER_QUICKSTART.md
